ledGrid
===============================

version number: 0.0.1
author: Daniel O Byrne

Overview
--------

A package to turn on and off lights on a grid

Installation / Usage
--------------------

To install use pip:

    $ pip install ledGrid


Or clone the repo:

    $ git clone https://github.com/obyrned1/ledGrid.git
    $ python setup.py install
